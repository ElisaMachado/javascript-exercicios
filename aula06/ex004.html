Aula 6 de Javascript / exercício 4

Aprendendo a formatar strings e números!

    STRINGS
  • var.length: para descobrir quantas letras tem na string;
  • var.toUpperCase(): para colocar toda a string em letras maiúsculas;
  • var.toLowerCase(): para colocar toda a strings em letras minúsculas.
    NUMBERS
  • var.toFixed(x): determinar o número de casas após a vírgula (colocar o número no lugar do x);
  • var.replace('.', ','): para substituir o ponto por vírgula;
  • var.toLocaleString('pt-br', {style: 'currency', currency: 'BRL'}): para colocar como dinheiro em reais;
    • Para outras moedas, trocar o 'BRL' para a sigla desejada.
      Ex: USD para dólar, EUR para euro, KRW para won, JPY para iene, etc.
